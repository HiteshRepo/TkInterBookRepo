# TkInterBookRepo

A program that stores these book information:
    Title, Author,
    Year and ISBN
    
User can do below operations:
    View all records
    Add an entry
    Search an entry
    update an entry
    delete an entry
    close
    
 DB Used : SQlite
 
 Py Modules Used : Tkinter, Sqlite3
